# Czech Language (by Hampap)

Tento jazykový balíček překládá hlavní části panelu.

_Některé novější sekce panelu mohou být ještě v angličtině; budeme je postupně doplňovat._

## Funkce

- Kompletní překlad hlavních částí rozhraní (dashboard, servery, profily, správa)
- Lokalizované systémové hlášky, tlačítka i formuláře
- Terminologie přizpůsobená českým hostingům a herním komunitám
- Snadné zapnutí přes nastavení jazyka v administraci

## Instalace

1. Nakopírujte složku pluginu `czech-language` do adresáře `plugins/`.
2. V administraci Pelican Panelu přejděte na **Settings → Plugins**.
3. Aktivujte plugin **Czech Language** a uložte změny.
4. V **Settings → Languages** vyberte češtinu jako výchozí jazyk panelu.
